SiGHT

A traversal solution for people with impaired sight and blindness

Comprised of a device that someone holds to give haptic feedback and guide them through their surroundings in the safest way possible. The device is paired an iOS application that will be available for iPhones and iPads, which leverages Apple’s AR Kit development environment that enables accurate 3D localization. Using LiDAR, this app allows us to get the data we need to direct people with blindness or low vision toward safe areas. With the app active, the phone is placed in a necklace-style case that the user can wear. From here, the phone talks to a sleeve-like attachment that we will design for the white cane which will vibrate and give signals to the user about their surroundings.

Donut PCs

PC parts and creation website made with Django and ReactJS

DP is a web application focused on PC parts and creation where we’ve developed a system so that customers can build and purchase computers and computer parts. The main set of actors involved are visitors, customers, employees, and owners. Visitors will simply be able to view the website and products, they cannot do anything the customers can do. The website will allow the customers to purchase, like, comment, create their own PCs, and have a financial system. Employees will be able to set up all this content, moderate it, and suggest builds. Owners are able to manage employees. Custom builds will focus on compatibility while pre-built builds will focus on user reviews.
